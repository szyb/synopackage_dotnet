Synopackage Proxy Repository

Synopackage Proxy Repository is an intermediary server between DSM and active source servers. It allows you to add a single repository link in your Package Center and get packages from all active repositories (or from some subset). All you need to do is to add the link to your Package Center.

Please note that this feature is still in beta phase.
